# 🚀 NexHire - Job Application Portal
<img width="1920" height="1080" alt="image" src="https://github.com/user-attachments/assets/c5f7fa3f-2888-420a-a103-cad1341c974b" />

NexHire is a full-stack MERN Job Portal application where students and recruiters can connect seamlessly.  
Users can register, login, browse jobs, apply for jobs, and manage profiles.

---

## 🌐 Live Features

- 🔐 Authentication & Authorization
- 👨‍🎓 Student & Recruiter Roles
- 📄 Job Posting & Applications
- 🏢 Company Management
- 👤 Profile Update with Image Upload
- ☁️ Cloudinary Image Storage
- 🍪 JWT Authentication with Cookies
- 📱 Responsive UI
- ⚡ Fast React + Vite Frontend

---

# 🛠️ Tech Stack

## Frontend
- React.js
- Vite
- Redux Toolkit
- Tailwind CSS
- Axios
- React Router DOM

## Backend
- Node.js
- Express.js
- MongoDB
- Mongoose
- JWT Authentication
- Multer
- Cloudinary
